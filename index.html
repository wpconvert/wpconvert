WP Convert

Isi judul, paste Featured Image, lalu isi artikel. Untuk gambar di isi artikel, cukup paste langsung di editor.

Input Konten

Paste gambar isi artikel langsung di editor
Klik area ini lalu paste Featured Image
Tekan Ctrl + V setelah copy gambar.
Belum ada Featured Image.
Kalau mau tambah gambar ke isi artikel, paste langsung di area editor putih di atas.

Kategori & Tags

Kategori bisa centang, tags isi manual
Pisahkan tags dengan koma, nanti otomatis masuk ke kolom Tags di WordPress.
Catatan:
- Featured Image dipakai untuk gambar tampilan depan.
- Gambar di isi artikel akan diupload otomatis saat dipaste ke editor.
- Tags dibuat / dipasang otomatis saat post.
